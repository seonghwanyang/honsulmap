## 1. 프로젝트 개요

### 핵심 문제
제주도 혼술바/게하 손님들은 이성 만남 목적으로 방문 → 가기 전에 사장님 인스타 스토리로 현황(남녀비율, 분위기 등) 확인 → **일일이 가게 인스타를 찾아보는 게 귀찮음**

### 솔루션
한번에 모아서 보여주는 사이트:
- 네이버맵 연동 (가게 위치)
- 인스타 스토리 자동 수집 or 링크 바로가기
- 유저 실시간 현황 게시 ("지금 여기 N명", 남녀비율 등)
- 사장님 홍보 게시판

### 기술 스택 (확정)
- **Next.js + Supabase + Vercel** (Sean 기존 스택)
- **네이버맵 API** (Sean이 네이버맵 연동 원함)
- 인스타: 무료 오픈소스 스크래핑

### 좌측: 기여랭킹 패널

- **카테고리 필터 (탭)**: role="tablist" — 전체 / 자유 / 투자 / 연애 / 꿀팁 / 직장인 / 학생. 우측에 검색 아이콘 버튼

- **게시글 리스트**: `<ul class="post-feed-list">` 무한스크롤 (sentinel 요소). 각 항목: 제목 + 카테고리 뱃지 + 작성자(랜덤 닉네임) + 댓글수(말풍선 SVG) + 찜수(hand-coins, lucide) + 시간(상대시간) + 이미지 첨부 뱃지
- 5번째 뒤 인라인 AdFit 728x90 (DAN-0j47gFJzpRVEd9iB)

### 우측: sticky 세로 광고

- 가성비 ⬆ / ⬇ 숫자
- 댓글수
- 시간 (상대시간)
- 썸네일 (우측, Supabase storage `deal-post-images/UUID/`)

## 9. 니 프로젝트 사이트 구조 (거지맵 기반 설계)

### 라우팅
- `/` or `/map` — 메인 지도 (혼술바/게하 핀 + 실시간 현황)
- `/feed` — 혼술러 커뮤니티 (후기/현황 피드)
- `/feed/:id` — 게시글 상세
- `/spot/:slug` — 가게 상세 (바텀시트, 인스타 스토리 + 현황 + 리뷰)
- `/promo` — 사장님 홍보/이벤트
- `/promo/:id` — 홍보 상세
- `/privacy` — 개인정보처리방침

### 메인(/map) 3컬럼
- **좌**: 인기 혼술바 랭킹 (방문자/현황 제보 수 기반), 지역 필터 칩 (제주시/애월/서귀포/동쪽/서쪽)
- **중앙**: 네이버맵 + 핀 (가게별 "지금 N명" or 분위기 이모지 🍻🔥😴), 핀 탭하면 → 바텀시트 오픈
- **우**: 최신 현황 피드 (심판대 역할, "누가 어디서 N명 제보함" 실시간)

### 가게 상세 (바텀시트, /spot/:slug)
- 가게명 + 주소
- 인스타 스토리 임베드/링크 + 가게 사진 스트립
- 메타: 카테고리(혼술바/게하), 영업시간, 전화번호
- 실시간 현황 타임라인 (유저 제보 히스토리)
- 액션: 찜 / 분위기⬆ / 네이버지도 / 분위기⬇
- 댓글 (닉네임+비번 익명 구조)

### 커뮤니티(/feed) 3컬럼
- **좌**: 현황 제보 기여랭킹 TOP50
- **중앙**: 게시글 피드 (카테고리 탭: 전체/현황/후기/꿀팁/자유), 인기글 캐러셀 pinned, 무한스크롤
- **우**: 광고 or 오늘의 추천 혼술바

### 사장님 홍보(/promo)
- 거지맵 핫딜 구조 차용
- 카테고리: 이벤트/할인/신규오픈/특별메뉴
- 가성비⬆/⬇ 대신 "가고싶다" 업다운
- 양쪽 세로 광고
